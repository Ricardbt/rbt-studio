# 🧠 RBT Studio AI Web

## 🎯 Skill Definition (AI Builder Context)

### Name

RBT Studio AI Website Generator

### Description

Build a modern, responsive website for a creative AI studio (RBT Studio) that showcases three main products: Brand AI, FlowonAI, and Storyprints. The site should be visually minimal, design-forward, and optimized for SEO and conversion.

### Capabilities

* Generate responsive React components
* Create SEO-friendly page structures
* Implement clean UI with modern design (grid, spacing, typography)
* Structure content into reusable sections
* Support dynamic routing (multi-page or SPA)

### Tech Stack

* React (18+)
* Vite or Next.js (preferred for SEO)
* TailwindCSS
* React Router (if not using Next.js)
