Motif rules

→ Always 1px or hairline stroke at <0.6 opacity. Never filled large.

→ Anchor on right or top-right of hero compositions. Never centered behind text.

→ Use a dotted or fine-grid backdrop. Never solid.

→ Sparkles ✦ are the only filled element — sparingly, max 3 per composition.
