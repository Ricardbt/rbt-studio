01

Principles

// Five rules. Test every decision against them.

01 / 05

Engineered, not decorated

Every element earns its place. If it has no function, remove it. Precision is the aesthetic.

02 / 05

Two colors are enough

Forest green on cream. Restraint signals confidence. Functional accents only when meaning demands it.

03 / 05

Sharp by default

Zero radius. Hairline borders. Editorial numbering. Round shapes are reserved for tags and the generative motif.

04 / 05

Code is image

Generative line-art accompanies hero moments. The system speaks fluent both technical and artistic.

05 / 05

Snappy motion

Confident, short, easy out. 180–240ms. Never bouncy, never sluggish. The interface responds before you blink.

05

Motion

// Snappy, confident, restrained. Motion communicates response — never decorates.

--dur-instant
100ms · hover, focus rings, button press feedback
--dur-quick
180ms · default UI · most state changes
--dur-base
240ms · panels, drawers, accordions
--dur-slow
420ms · page transitions, hero reveals

--ease-snap

cubic-bezier(0.2, 0.7, 0.1, 1)

Default. Quick exit, confident landing. The studio's signature curve.

--ease-out

cubic-bezier(0.16, 1, 0.3, 1)

Decisive arrivals. Modal opens, page transitions, hero reveals.

--ease-in

cubic-bezier(0.7, 0, 0.84, 0)

Anticipation. Dismissals and elements leaving the viewport.

06

Voice

// Bilingual (ES/EN). Direct sentences. Technical when accuracy matters; warm when humans are listening.

✦   Do

"Aplicaciones web de producción. Desde MVPs hasta plataformas escalables."

"Ingeniería de precisión y sensibilidad artística."

"Proyectos que conectan el mundo físico y digital."

"Disponible globalmente en remoto. Respuesta <24h."

×   Don't

"We are passionate about leveraging cutting-edge solutions to disrupt..."

"🚀 Take your business to the next level! 💪"

"Innovative, world-class, best-in-breed AI synergies."

"Drop us a line and let's grab a coffee!! ☕"

Specific

Name the tools, not the buzzword. React, n8n, Three.js — not "AI-powered solutions".

Bilingual

Spanish primary, English secondary. Both in production. Catalan welcomed.

Restraint

No exclamation marks. Maximum one emoji per page (✦). Periods do the work.

Numbered

Lists are 01 / 02 / 03. Editorial. Ordered. Confident.
